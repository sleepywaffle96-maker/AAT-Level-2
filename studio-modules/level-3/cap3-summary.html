MATC Formulas & Revision Core

Mathematical core formulas for short-term control and inventory evaluation metrics.

1. The High-Low Calculations

Variable Cost Per Unit (VCPU)
VCPU = (Highest Cost - Lowest Cost) / (Highest Activity - Lowest Activity)
Fixed Cost Element
Fixed Cost = Total Cost - (VCPU x Total Units Produced)

2. Overhead Metrics

Overhead Absorption Rate (OAR)
OAR = Budgeted Overheads / Budgeted Activity Level (Hours/Units)

3. Marginal Analysis Metrics

Contribution Per Unit
Contribution = Selling Price - Variable Costs
Break-Even Point (Units)
BEP = Total Fixed Costs / Contribution Per Unit

4. Profit Reconciliation Rule

  • Stock Increases: Absorption Profit is higher than Marginal Profit.
  • Stock Decreases: Marginal Profit is higher than Absorption Profit.
  • Reconciliation Formula: Difference in Profit = Change in Stock Units x Fixed OAR per unit.
