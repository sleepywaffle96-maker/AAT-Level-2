DAIF Consolidated Formula Cheat Sheet

Essential mathematical core workflows for group accounts consolidation.

1. Provisions for Unrealised Profit (PUP Formulas)

PUP via Profit Mark-up Percentage
PUP = Value of Remaining Intra-group Stock x (Mark-up% / 100)
PUP via Gross Profit Margin Percentage
PUP = Value of Remaining Intra-group Stock x (Margin% / 100)

Double Entry: Debit Group Retained Earnings (Selling Entity Share) and Credit Consolidated Inventory.

2. Statutory Goodwill Balance (IFRS 3)

Goodwill = Parent Consideration + NCI Valuation - Fair Value of Subsidiary Net Assets at Acquisition

3. Closing NCI Balance Working

Closing NCI = NCI Fair Value at Acquisition + (NCI % x Subsidiary Post-Acquisition Profit)

4. Consolidation Exam Core Rules

  • Line-by-Line Consolidation: 100% of all subsidiary asset and liability accounts are added directly to the parent metrics, regardless of whether ownership is below 100%. NCI accounts for the difference.
  • Pre-Acquisition Retained Earnings: The exact value of a subsidiary's reserves on the day it was bought. This figure is frozen and used only to calculate goodwill; it can never be distributed as group profit.
  • Intra-group Dividends: Payments sent from a subsidiary to its parent must be eliminated entirely from the consolidated financial statements.
